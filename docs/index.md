プライバシーポリシー
本プライバシーポリシーは、[Meitel]（以下、「開発者」といいます）が提供するスマートフォン用アプリ「Resonance AI」（以下、「本アプリ」といいます）における、ユーザーの個人情報およびプライバシー情報の取り扱いについて定めるものです。
取得する情報および利用目的
本アプリでは、以下の情報を取得し、それぞれの目的で利用します。
・ユーザーが入力したテキストデータ
利用目的：AIによる応答（アドバイス）を生成するため。
・音声データ
利用目的：音声認識機能（SpeechRecognizer）を利用し、音声をテキストに変換するため。音声データ自体は保存されません。
・アプリの利用状況（幸せの共有件数など）
利用目的：アプリ内の「世界との共鳴」機能において、全体の共有数をカウントおよび表示するため。
外部サービスの利用について
本アプリでは、サービスの提供にあたり以下の外部サービスを利用しており、各サービスのプライバシーポリシーに基づいてデータが処理されます。
・Google Gemini API
入力されたテキストは、AIによる応答生成のためにGoogleのサーバーへ送信されます。
・Firebase (Google Cloud)
共有数などのアプリ内データは、安全に同期・保存するためにFirebase Firestoreを利用しています。
・Google 音声認識 (Speech Services by Google)
音声入力機能を利用する際、音声データはテキスト変換のためにGoogleのサーバーで処理されます。
